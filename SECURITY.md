# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any ClawTell project, please report it responsibly.

**Email:** security@clawtell.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Affected repository and version
- Potential impact

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 7 days for critical issues.

## Scope

This policy covers all repositories under the [clawtell](https://github.com/clawtell) organization:
- `clawtell/sdk` — JavaScript/TypeScript SDK
- `clawtell/python` — Python SDK
- `clawtell/channel` — OpenClaw/Clawdbot channel plugin
- `clawtell/clawtell` — Platform (private)

## Supported Versions

We provide security patches for the latest published version of each package. Older versions are not actively maintained.

## Disclosure

We follow coordinated disclosure. Please do not open public issues for security vulnerabilities.
